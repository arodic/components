## `IoColorPicker`

Extends `IoColorMixin(IoItem)`.

Implements `IoColorSwatch`, `IoColorPanelSingleton` and `IoLayerSingleton`.

Input element for color picking. Expands a floating color panel when clicked or activated by keyboard.
